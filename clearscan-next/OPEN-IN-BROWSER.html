Industrial fruit ripeness comparison using AI

This is a preview. The full app (camera, scan, Clerk auth) runs with Node.js.

To run the full app

Node.js is required. Then start the server and open http://localhost:3000

  1. Install Node: nodejs.org (download LTS, run the installer)
  2. In Terminal, run: cd /Users/apple/Desktop/FruityVisionAI/clearscan-next && npm install && npm run dev
  3. Or double‑click START-HERE.command in this folder (it will try to install Node via Homebrew if needed)
